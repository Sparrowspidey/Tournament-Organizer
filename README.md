# Tournament Organizer

A Java app that lets users create and manage **league** and **elimination-style** tournaments, generate match schedules, and export brackets to PDF using iTextPDF.

---

## Features
- Create and manage **league-style tournaments** (round-robin)  
- Create and manage **elimination-style tournaments** (single-elimination)  
- Add and manage teams  
- Automatically generate match schedules or brackets  
- Export schedules/brackets to PDF using iTextPDF  
- Console-based, menu-driven UI  

---

## Requirements
- Java 8 or higher  
- Maven (recommended) or manual `.jar` management for dependencies  
- iTextPDF library (if not using Maven)  

---

## Using JARs (Manual)

- Download itextpdf-5.5.13.3.jar and put it in a libs/ folder
